# java-web-sbt-spring-jsp-hello-world

## Description
A springboot web app with jsp support.

#W Tech stack
- java
  - springboot
  - jsp
- sbt

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`
[Available at](http://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
